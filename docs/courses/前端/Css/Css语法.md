## 注释
注释是用来解释你的代码，并且可以随意编辑它，浏览器会忽略它。<br>
CSS注释以 /* 开始, 以 */ 结束, 实例如下:
```css
/*这是个注释*/
p{
    text-align:center;
    /*这是另一个注释*/
    color:black;
    font-family:arial;
}
```
