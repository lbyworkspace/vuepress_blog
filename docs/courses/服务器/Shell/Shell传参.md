我们可以在执行 Shell 脚本时，向脚本传递参数。<br>
脚本内获取参数的格式为：$n。n 代表一个数字，1 为执行脚本的第一个参数，2 为执行脚本的第二个参数，以此类推……<br>
$0为执行的文件名（包含文件路径）
```sh
echo "执行的文件名：$0";
echo "第一个参数为：$1";
echo "第二个参数为：$2";
echo "第三个参数为：$3";

./test.sh 1 2 3

# 执行的文件名：./test.sh
# 第一个参数为：1
# 第一个参数为：2
# 第一个参数为：3
```
